### Almost a Spotify 
Consumindo API do Spotify para buscar álbums e as faixas neles

### Use
- `git clone https://github.com/PedroVictor197/client-spotify.git`
- `cd client-spotify`
- `npm i`
- `npm start`
- `cd server`
- `npm run devStart`

Se o seu navegador não abrir automaticamente va até `http://localhost:3000`
